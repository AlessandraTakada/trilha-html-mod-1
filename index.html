Citações

A tag <blockquote> indica uma citação longa e é caracterizada pela inserção de um recuo.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ab minus perferendis nostrum praesentium, aut, beatae illo soluta asperiores quisquam labore quos numquam at fugiat quaerat laboriosam ducimus, esse veniam inventore. Alias similique recusandae ipsa provident nulla. Unde veritatis sint ex? Voluptatum architecto quas quisquam est fugit nesciunt doloribus culpa perspiciatis harum quibusdam nostrum deleniti, libero laboriosam aut nam, voluptas exercitationem?

A tag <q> indica uma citação curta e é caracterizada pela inserção de aspas.

Exemplo:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Laudantium consequatur ipsum, amet, eum incidunt porro exercitationem reprehenderit animi excepturi ex tenetur dolores sunt praesentium eaque quam autem, molestiae reiciendis pariatur. Ipsum perferendis deleniti deserunt perspiciatis quo harum atque quod. Eos saepe blanditiis sequi neque obcaecati asperiores nisi sed odit incidunt placeat a qui, voluptatibus sapiente eveniet commodi aliquid.

Listas

A tag <ol> indica uma lista de itens ordenados.

Exemplo:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

A tag <ul> indica uma lista de itens não ordenados.

Exemplo:

  • Item 1
  • Item 2
  • Item 3

Tabelas

A tag <table> define uma tabela.

Os conteúdos de uma tabela podem ser agrupados da seguint forma:

  • <thead>: agrupa elementos do cabeçalho de uma tabela
  • <tbody>: agrupa elementos do corpo de uma tabela
  • <tfoot>: agrupa elementos do rodapé de uma tabela

A tag <th> define um cabeçalho.

A tag <tr> define uma linha.

A tag <td> define uma célula de dados.

Fornecer uma descrição do propósito da tabela é importante para questões de acessibilidade e é feito pela tag <caption>

Formulários

A tag <form> define um formulário.

A tag <input> cria um campo de entrada para que o usuário insira algum dado que pode ser do tipo text, password, number, entre outros.

A tag <textarea> cria uma caixa de texto para inserção de texto longo, como uma caixa de comentários, por exemplo.

A tag <label> define o rótulo de cada input.

A tag <button> define um botão clicável, que pode, por exemplo, receber o atributo submit responsável por enviar os dados do formulário

Hiperlink

A tag <a> permite a hiperligação para páginas web, endereços de e-mail, ancoragem na mesma página, arquivos. Deve ser acompanhada do atributo href indicando o destino do link.

Multimídia
